\subsection{Detailed Description}
Memory allocation functions. 

Provides the functions to allocate a 1\+D, 2\+D, or 3\+D array as a contiguous block of memory. For 2d (3\+D) arrays, the right most index (i.\+e., ni here) runs over contiguous memory chunks, followed by nj, and nk (if defined).
